# Frontend Components

A collection of UI components and projects built to practice and strengthen front-end development skills on my journey to becoming a front-end developer.  

---

## Goals

This repository is a living record of my growth as a front-end developer. My aim is to work through a broad range of projects and technologies, including:

- Writing clean, semantic HTML
- Building responsive layouts with CSS
- Creating reusable UI components
- Adding interactivity with JavaScript
- Exploring modern front-end frameworks and libraries
- Strengthening front-end fundamentals from the ground up

## Running the Projects Locally

1. Clone the repository
2. Navigate into `html-css`, `html-css-js`, or `vue`
3. Open the desired project folder
4. Open `index.html` in your browser (for Vue, run `npm run dev` or serve the `dist/` folder)
