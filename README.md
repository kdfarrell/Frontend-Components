# Frontend Components

A collection of UI components and small projects built to practice and strengthen front-end development skills using **HTML, CSS, and JavaScript**.  
Projects are organized by technology and completion order, making it easy to track progress and revisit earlier work.

---

## Goals

This repository focuses on:

- Writing clean, semantic HTML  
- Building responsive layouts with CSS  
- Creating reusable UI components  
- Adding interactivity with JavaScript  
- Strengthening front-end fundamentals

## Running the Projects

To view any project locally:

1. Clone the repository  
2. Navigate into either the `html-css` or `html-css-js` folder  
3. Open the desired project folder  
4. Open the `index.html` file in your browser
